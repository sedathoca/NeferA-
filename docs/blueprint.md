# **App Name**: Classroom Ace

## Core Features:

- School & Class Setup: Capture school, class, and homeroom teacher information.
- Student Import: Import student list from a file (e.g., CSV).
- Seating Arrangement - Drag and Drop: Arrange student seating via a drag and drop interface, in a traditional row/column format.
- Automatic Seating Assistant: Suggest optimal seating arrangements based on criteria (e.g., keeping students who struggle apart). This feature will use an AI tool.
- Export to Document: Export the seating arrangement to Word and PDF formats.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to create a sense of order and calm. This evokes trustworthiness, productivity and organization.
- Background color: Very light blue (#E3F2FD), visibly desaturated version of the primary color.
- Accent color: Soft green (#A5D6A7) for positive actions.
- Body and headline font: 'PT Sans', a humanist sans-serif with a little warmth.
- Use simple, clear icons to represent actions and elements in the interface.
- A clean, well-spaced layout to aid in readability and ease of use.
- Subtle transitions and animations to enhance the user experience and provide feedback during interactions.
